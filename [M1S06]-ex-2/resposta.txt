[M1S06] Ex 2 - Verbos HTTP
---------------------------

Associe a seguir qual verbo HTTP corresponde com a afirmativa:

1. GET
2. POST
3. PATCH
4. PUT

Resposta:

* Verbo HTTP responsável por capturar dados muito utilizado para listar dados vindo do backend; (1) ==> GET
* Verbo HTTP utilizado para alterar apenas um de seus elementos normalmente envia-se um parâmetro pela URL; (3) ==> PATCH
* Verbo HTTP utilizado para alterar todos os seus elementos; (4) ==> PUT
* Verbo HTTP utilizado para enviar dados no corpo da requisição, responsável muitas vezes pelo cadastro de novos itens; (2) ==> POST
